üìà How Optimization Works

The Process:

  1. Data Collection: System records every trade with entry score, exit price, profit/loss
  2. Weight Optimization: Uses scipy.optimize to find weights that maximize Sharpe ratio
  3. Threshold Testing: Tests thresholds 50-85 to find optimal trade-off between quantity and quality
  4. Recommendation: Suggests changes that improve risk-adjusted returns
  5. Manual Update: You review and update config/config.py
  6. Bot Restart: Bot uses new settings for future trades
  7. Continuous Improvement: Re-run optimizer weekly as more data accumulates

Why It Works:

  • Historical Evidence: Recommendations based on YOUR actual trading results
  • Statistical Significance: Requires 30+ trades for reliable optimization
  • Risk-Adjusted: Optimizes Sharpe ratio, not just profit (controls for volatility)
  • Safe: You review all changes before applying
